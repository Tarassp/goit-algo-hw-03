# goit-algo-hw-03

Домашнє завдання до модуля 3. Робота з датами, генерація випадкових чисел та обробка рядків.

## Завдання

### task_1.py — `get_days_from_today(date)`

Функція приймає рядок дати у форматі `YYYY-MM-DD` та повертає кількість днів між переданою датою і поточною.

    get_days_from_today("2021-10-09")  # -> 1666

### task_2.py — `get_numbers_ticket(min, max, quantity)`

Функція генерує відсортований список унікальних випадкових цілих чисел у діапазоні `[min, max]`. Повертає порожній список, якщо параметри некоректні.

    get_numbers_ticket(1, 49, 6)  # -> [4, 15, 23, 31, 40, 47]

### task_3.py — `normalize_phone(phone_number)`

Функція нормалізує рядок з номером телефону до формату `+38XXXXXXXXXX`, видаляючи зайві символи та додаючи код країни за потреби.

    normalize_phone("(050)8889900")  # -> "+380508889900"
